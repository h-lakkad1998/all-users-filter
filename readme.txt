== Description ==
Plugin for filtering, sorting, and exporting users.

This plugin allows you to filter, sort, and export users in CSV format. You can filter users by multiple parameters, such as date, role, meta key-value, and registration date. 

Developed by Hardik Patel (also known as Hardik Lakkad). For more information, visit [LinkedIn Profile](https://www.linkedin.com/in/hardik-lakkad-097b12147/).

== Installation ==

To install the plugin, follow these steps:

1. Go to the Plugins page.
2. Click the Add New button.
3. Search for the plugin name.
4. Click the Install Now button.
5. Click the Activate button.

The plugin is now installed. Note: Only administrators can use the plugin.
